### Что нужно сделать перед запуском:

1. **Получить токен бота:**
   - Зайти к @BotFather
   - Создать нового бота
   - Скопировать токен

2. **Настроить .env:**
   ```bash
   cp .env.example .env
   # Отредактировать .env
   ```

3. **Добавить контент:**
   - Поместить видео в `content/media/videos/`
   - Поместить документы в `content/media/documents/`

4. **Запустить:**
   ```bash
   # Локально
   python bot.py
   
   # Docker
   docker-compose up -d --build
   ```
